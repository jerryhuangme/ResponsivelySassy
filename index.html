ResponsivelySassy

A responsive multi-column framework built on SaSS

Hello world! This is ResponsivelySassy with HTML5 boilerplate. Change your browser window to see this script in action.

A light weight responsive CSS framework built for simplicity and adaptability. Using SaSS CSS, two configuration variables are required: maxpagewidth and columns decided. Once these two variables are defined, ReponsivelySassy will automatically calculate gutter, width, padding and compute media queries for all view ports.
